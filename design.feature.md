### 키워드 조회 기능
1. 단순 키워드 조회
    - 명령어 : [키워드]
    - 응답 메시지 : 
    ```
    # [키워드] 검색량 입니다! 
    
    # 블로그 문서량 : 100,240
    # ToTal 검색량 : 55,990
    # PC 검색량 : 7,590
    # MOBILE 검색량 : 48,400
    
    # 데이터 통계 기간
    2017.08.05 ~ 2017.09.05
    ```

2. 복수 키워드 조회
    - 명령어 : [키워드1, 키워드2, 키워드3]
    - 응답 메시지 : 
    ```
    ——————————————————————————
    🤪자비스 두뇌 200% 풀가동!!🤪
    ——————————————————————————

    ————————————
    # [부스트캠프] 검색량 입니다 !

    # 블로그 문서량 : 100,240
    # ToTal 검색량 : 55,990
    # PC 검색량 : 7,590
    # MOBILE 검색량 : 48,400
    ————————————

    ————————————
    
    # 데이터 통계 기간
    2017.08.05 ~ 2017.09.05

    ```
